Ejercicio 2 - API Testing con Karate DSL

Tecnologías usadas:
- Karate DSL 1.4.1
- Java 17
- Maven 3.9+
- IntelliJ IDEA (opcional)

Pasos para ejecutar:
1. Clonar el repositorio.
2. Ejecutar los tests con:
   mvn test

Estructura:
- get-products.feature: valida producto estrella.
- get-category-products.feature: valida productos por categoría.

Requisitos:
- Tener Java y Maven instalados.
- Conexión a internet para acceder a la API pública.
